# MLOpsAssignment
MLOps assignment from the Applied AI & Agentic AI program at IIITB with upGrad — covering model lifecycle, CI/CD pipelines, and deployment practices.

# Python Version
Python 3.11 was selected because it provides strong performance improvements, modern language features, and stable compatibility with the required machine learning and MLOps libraries such as scikit-learn, MLflow, XGBoost, and LightGBM. It offers a good balance between modern runtime optimizations and ecosystem stability for production-oriented ML workflows.
